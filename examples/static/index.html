@fastnear examples

Static HTML demos — no build tools, just <script> tags

View Only

Read on-chain data with near.view() — no wallet needed. Fetches wNEAR token metadata from mainnet.

mainnet

Greeter

Read + write with wallet sign-in. Calls greeter.testnet to get/set a greeting, plus NEP-413 message signing.

testnet wallet

Berry Club

Session keys, full-access transactions, and a live pixel board. Draws on berryclub.ek.near on mainnet.

mainnet wallet session keys
